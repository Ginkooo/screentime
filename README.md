A minimal screentime deamon

# Install:

`cargo install`

# Usage:
1. Run `screentime`
2. Make a `GET` request to `http://127.0.0.1` and it will respond you with your today's screentime

# Features:

- [x] Linux, Windows and MacOS
- [x] Messure total screentime
- [x] AFK feature
- [ ] Configurable
- [ ] Simple client in a binary
